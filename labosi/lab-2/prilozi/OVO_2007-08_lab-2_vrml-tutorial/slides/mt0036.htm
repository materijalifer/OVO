Building primitive shapes
Syntax: Cylinder

  • A Cylinder geometry node builds an upright cylinder
  • height and radius - cylinder size
  • bottom, top, and side - parts on or off


    • [ cyl.wrl ] 
    Shape {
    appearance Appearance {
        material Material { }
    }
    geometry Cylinder {
        height 2.0
        radius 1.0
        bottom TRUE
        top TRUE
        side TRUE
    }
}
