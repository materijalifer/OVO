Adding sound
Syntax: AudioClip

  • An AudioClip node creates a digital sound source
  • url - a sound file URL
  • pitch - playback speed
  • playback controls, like a TimeSensor node 

    • Sound {
    source AudioClip {
        url "myfile.wav"
        pitch 1.0
        startTime 0.0
        stopTime  0.0
        loop FALSE
    }
}
