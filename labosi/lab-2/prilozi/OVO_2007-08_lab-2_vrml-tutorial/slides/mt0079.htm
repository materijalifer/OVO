Controlling appearance with materials
Appearance example code

Shape {
    appearance Appearance {
        material Material {
            diffuseColor  0.2 0.2 0.2
            emissiveColor 0.0 0.0 0.8
            transparency  0.25
        }
    }
    geometry Box {
        size 2.0 4.0 0.3
    }
}
