Controlling color on coordinate-based geometry
Syntax: PointSet

  • A PointSet geometry node creates geometry out of points
  • color - provides a list of colors
  • Always binds one color to each point, in order 

    • Shape {
    appearance Appearance { . . . }
    geometry PointSet {
        coord Coordinate { . . . }
        color Color { . . . }
    }
}
